# 🚀 CRASH.BET - Ultimate Crash Betting Experience

![Crash Bet Hero](https://images.unsplash.com/photo-1614028674026-a65e31bfd27c?auto=format&fit=crop&q=80&w=1000)

Welcome to **CRASH.BET**, a high-stakes, real-time crash betting platform designed with a sleek dark neon aesthetic. Experience the thrill of the multiplier as the plane takes flight!

## ✨ Key Features

- **🎰 Multi-Slot Betting**: Place up to **5 bets simultaneously** in a single round. Manage each slot independently with its own amount and auto-cashout settings.
- **⚡ Real-Time Gameplay**: Silky smooth 60FPS canvas animations powered by high-performance WebSockets.
- **🛡️ Provably Fair**: Every round's outcome is mathematically verifiable. We use HMAC SHA-256 with server/client seeds to ensure transparency.
- **👑 Admin Command Center**: A powerful dashboard for admins to manage user balances, grant coins, and monitor activity with ease.
- **🤖 Auto-Cashout**: Define your exit strategy. Set a target multiplier, and the system will automatically secure your wins.
- **⏳ Next-Round Queueing**: Don't miss a beat. Place your bets for the next flight while the current one is still in the air.

## 🎮 How to Play

1. **Sign Up**: Create an account in seconds.
2. **Get Funded**: Contact an administrator to have coins added to your balance.
3. **Configure Your Bets**: Choose your amount for any of the 5 available slots.
4. **Take Off**: Watch the multiplier climb. The higher it goes, the more you win!
5. **Secure the Bag**: Click the **CASH OUT** button before the plane disappears.

## 🔍 Provably Fair System

We believe in 100% transparency. Each crash point is derived from:
- **Server Seed**: A private, random 256-bit string generated on the server.
- **Client Seed**: A public hash that updates daily.
- **Nonce**: A unique counter for every round.

The final crash point is calculated using a standard HMAC-SHA256 algorithm, making it impossible for the house to manipulate results after a round has started.
